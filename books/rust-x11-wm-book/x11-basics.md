## X Window System

このセクションでは、X Window Systemの概要とクライアント・サーバーモデルについて説明します。

X Window Systemは、Unix-likeなOSでグラフィカルなアプリケーションを動かすためのシステムです。1980年代にMITで開発され、現在は [X.org Foundation](https://x.org/wiki/) で開発されています。

X Window Systemは、XやX11と呼ばれることもあります。X11の11は、プロトコルのバージョン番号です。本書では、X11と表記します。

:::message
近年は、X11に代わるWaylandという技術も登場していますが、本書ではX11を扱います。
:::

X11は、クライアント・サーバーモデルを採用しています。このモデルでは、XサーバーとXクライアントが役割を分担します。

Xサーバーは、クライアントからのリクエストを受け取って画面に表示したり、キーボードやマウスの入力をクライアントに送ったりするプログラムです。代表的な実装として Xorg があります。

Xクライアントは、ターミナルやブラウザなどのアプリケーションです。Xサーバーにリクエストを送ってウィンドウを表示させたり、サーバーから入力を受け取ったりします。

クライアントとサーバーは、X11 Protocolに従って通信します。X11 Protocolは、通信のフォーマットを定めた仕様です。

X11のアプリケーション開発では、X11 Protocolを使いやすくするライブラリを使います。C言語では XCB や Xlib が使われます。本書では、Rust向けのライブラリである x11rb を使用します。
