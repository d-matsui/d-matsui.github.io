## X Window System

このセクションでは、X Window Systemがどのようなシステムであるかを説明し、X11, X Display Server, Xlibなどの用語を整理します。これにより、次章以降の実装例を読み解くための基礎知識を提供します。

X Window Systemは、Unix-likeなOSにおいて、画面への描画やマウス・キーボードの入力を扱うためのシステムです。1980年代にMITで開発され、現在は [X.org Foundation](https://x.org/wiki/) で開発されています。

X Window Systemは、XやX11と呼ばれることもあります。X11の11は、プロトコルのバージョン番号です。本書では、X11と表記します。

X11は、クライアント・サーバーモデルを採用しています。サーバーは、画面・キーボード・マウスなどのハードウェアを管理し、クライアントからのリクエストを処理します。クライアントとサーバーは、X11 Protocolに従ってリクエストやリプライを送受信します。

X11に関連する主な用語を以下に示します。

- X11 Protocol: クライアントとサーバー間の通信ルールを定めた仕様
- Xサーバー (X Display Server): X11 Protocolを実装したサーバー (代表例: Xorg)
- Xクライアント: Xサーバーに接続するクライアントアプリケーション
- クライアントライブラリ: X11 Protocolをラップしたライブラリ (XCB, Xlib, x11rbなど)

X11のアプリケーションを開発する際、X11 Protocolの詳細を理解して実装を進めることは大変です。そのため、XCBやXlibのようなライブラリが存在します。本書では、RustでX11を扱うためのライブラリである x11rb を使用します。
